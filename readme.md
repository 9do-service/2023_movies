## 2023 台灣票房

專案以 http server 的方式啟動，執行專案的 start.bat 啟動一個本地網站伺服器，請不要關閉啟動的伺服器。

在瀏覽輸入 http://127.0.0.1:8000/index.html 開啟瀏覽器查看資料內容
